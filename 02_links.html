HTML Link Element

Anchor Tag is used for link element in HTML.

It is denoted by <a>

<a href="url link to be placed in here" > Link Text </a>

href attribute

href attribute is used to place the link to be navigated to by the user when on clicking the link text.
For example, href=https://google.com

NOTE:
Always include https:// when adding the url. It is necessary to include when navigating to the desired url.

target attribute

target attribute is used to open the desired in the current tab or a new tab depending on the value given to it.
for example,

  1. default target value:
    target attribute is optional, it value not declared in the <a> tab then the url opens in the current tab only.
    Example: Click on the link below to check.
    Google

  2. _blank target value:
    _blank value is given to target attribute when user wants to open the url in the new tab.
    Example: Click on the link below to check.
    W3Schools
