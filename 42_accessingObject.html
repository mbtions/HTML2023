Accessing Objects in JavaScript

Two methods:

  1. Dot Notation
  2. Bracket Notation
        const obj = {
            name: "meenakshi",
            course: "MCA",
            college: "KIET",
            rollno: 29
        }


        // dot Notation
        obj.name; // "meenakshi"
        obj.rollno; // 29

        // bracket Notation
        let value = course
        obj[value]; // "MCA"
        obj["name"]; // meenakshi
